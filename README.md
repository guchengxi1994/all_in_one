# all_in_one

### how to run

* create a `.env` file in root folder with configs like
  ```
    LLM_BASE = LLM API
    LLM_MODEL_NAME = model name
    CHAT_CHAT_BASE = http://x.x.x.x:7861
    LLM_SK = your secret key
  ```
